# [JS] 표현식과 연산자 📝

<br />

## **할당 연산자 💭**

- `x 연산자 y`

| 연산자   | 설명                                 | 비고                   |
| -------- | ------------------------------------ | ---------------------- |
| =        | 할당                                 | x = y                  |
| +=       | 더하기 할당                          | x = x + y              |
| -=       | 빼기 할당                            | x = x - y              |
| \*=      | 곱하기 할당                          | x = x \* y             |
| /=       | 나누기 할당                          | x = x / y              |
| %=       | 나머지 할당                          | x = x % y              |
| \*\*=    | 거듭제곱 할당                        | x = x \*\* y           |
| <<=, >>= | 왼쪽 시프트 할당, 오른쪽 시프트 할당 | x = x << y, x = x >> y |
| >>>=     | 부호 없는 오른쪽 시프트 할당         | x = x >>> y            |
| &=       | 비트 AND 할당                        | x = x & y              |
| ^=       | 비트 XOR 할당                        | x = x ^ y              |
| \|=      | 비트 OR 할당                         | x = x \| y             |
| &&=      | 논리 AND 할당                        | x && (x = y)           |
| \|\|=    | 논리 OR 할당                         | x \|\| (x = y)         |
| ??=      | 널 병합 할당                         | x ?? (x = y)           |
|          |                                      |                        |

<br />

## **비교 연산자 💭**

| 연산자 | 설명                             | 비고 |
| ------ | -------------------------------- | ---- |
| ==     | 같으면 참                        |      |
| !=     | 다르면 참                        |      |
| ===    | 값과 타입이 모두 같은 경우 참    |      |
| !==    | 갑 또는 타입이 서로 다를 경우 참 |      |
| >, <   | 크거나 작으면 참                 |      |
| <=, => | 크거나 같으면, 작거나 같으면 참  |      |
|        |                                  |      |

<br />

## **산술 연산자 💭**

| 연산자 | 설명                                   | 비고 |
| ------ | -------------------------------------- | ---- |

| --     | 감소                                   |      |
| -      | 빼기, 부호를 반대로 바꿈               |      |

| \*\*   | 거듭제곱                               |      |

| \*     | 곱하기                                 |      |
|        |                                        |      |
