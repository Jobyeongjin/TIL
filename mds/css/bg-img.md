# [CSS] 배경 이미지 만들기

### 이미지 속성 코드

```css
.intro_bg {
  background-image: url("이미지 주소입력");
}
```

<br />

### 함께 쓰면 좋은 배경 속성은 ? ☺️

```css
.intro_bg {
  background-repeat: no-repeat; /* 사진 반복은 그만 */
  background-size: cover; /* 배경을 꽉차게 */
}
```

<br />
