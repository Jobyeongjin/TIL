# [Python] 델타 탐색

## **이차원 리스트의 원소 탐색하기**

```python
arr = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
    ]

# 상 하 좌 우
row = [-1, 1, 0, 0]
col = [0, 0, -1, 1]


# 5를 기준 좌표로 설정
r = 1
c = 1

for i in range(4):
    x = r + row[i]
    y = c + col[i]

    print(arr[x][y])

>>> 2, 8, 4, 6


# 2를 기준 좌표로 설정
r = 0
c = 1

for i in range(4):
    x = r + row[i]
    y = c + col[i]

    print(arr[x][y])

>>> 8, 5, 1, 3
```
